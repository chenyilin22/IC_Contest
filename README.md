# 全國大學校院積體電路(IC)設計競賽 - 標準元件數位電路組
本專案整理「全國大學校院積體電路(IC)設計競賽－標準元件數位電路組」歷年競賽題目的解題實作紀錄。內容包含 Verilog  撰寫、模擬驗證、時序優化等，適合對數位邏輯設計與比賽有興趣的學生或初學者參考學習。
